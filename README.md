# 🎮 Tetris Game
A browser-based implementation of the classic Tetris game built using JavaScript.

## ✨ Features
* Classic Tetris gameplay
* Keyboard controls for movement and rotation
* Score tracking
* Grid-based game logic

## 🛠️ Tech Stack
* HTML
* CSS
* JavaScript

## 📌 What I Learned
* Game logic implementation
* Handling user input (keyboard events)
* Managing state (tracking things like piece position, score, and grid) and updating the screen accordingly

## ⚙️ How to Run
1. Clone the repository
2. Open `index.html` in your browser
